# vscode-colab-integration

This project integrates Google Colab with Visual Studio Code, allowing users to leverage the capabilities of Colab directly within their development environment.

## Features

- Seamless connection to Google Colab
- Easy authentication with Google services
- Command palette integration for Colab features
- Example notebook to demonstrate functionality

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/vscode-colab-integration.git
   ```

2. Navigate to the project directory:
   ```
   cd vscode-colab-integration
   ```

3. Install the required extensions:
   - Open the `.vscode/extensions.json` file and install the recommended extensions.

4. Install the project dependencies:
   ```
   npm install
   ```

## Usage

1. Open the command palette (Ctrl+Shift+P) and search for "Colab".
2. Select the desired command to interact with Google Colab.
3. Follow the prompts for authentication and usage.

## Example

An example Jupyter notebook is provided in the `notebooks` directory. Open `example.ipynb` to see how to use the integration.
